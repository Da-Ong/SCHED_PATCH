# SCHED_PATCH
 
**`Enlish`** | [简体中文](README.md)
 
Unofficial Fengchi patch repository. Since the official Fengchi kernel has not been open-sourced, this repository organizes leaked source code and produces patches that can be applied during kernel compilation.
